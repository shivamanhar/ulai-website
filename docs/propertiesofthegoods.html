Свойства товаров

Шаг 1

Чтобы создать новое свойство или редактировать уже существующее, необходимо:

Каталог товаров → Свойства товаров

Шаг 2

Для создания нового свойства

Клик по кнопке “Создать свойство”

Для редактирования

Клик по названию свойства в столбце “Свойство”

Шаг 3

Указать или изменить содержимое полей:

  • Имя
  • Имя колонки CSV
  • Использовать в категориях
  • Значение
  • А также осуществить настройку создаваемого свойства в блоке ячеек

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти” в правом верхнем углу экрана

Шаг 4

Очистите кеш интернет-магазина, осуществив переход: Настройки → Очистить кеш:

Если вам нужно заполнить только основные данные без расширенных настроек, рекомендуем попробовать функцию быстрого наполнения - быстрое создание свойства товара
