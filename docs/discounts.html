Скидки

По умолчанию создана только одна скидка - скидка для администратора.

Чтобы создать/редактировать скидку, необходимо:

Шаг 1

В панели администратора осуществить переход: Модули → Скидки интернет-магазина

Шаг 2

Для создания скидки - осуществить клик по кнопке “Создать скидку”

Для редактирования скидки - осуществить клик по названию скидки в столбце “Ключ”

Шаг 3

Далее в открывшейся форме создания/редактирования скидки необходимо:

Для создания скидки:

  1. Указать название скидки
  2. При необходимости установить ограничение на количество использований
  3. Выбрать метод подсчета скидки
  4. Выбрать тип скидки
  5. При необходимости указать временной интервал действия скидки

Для редактирования (при необходимости) скидки:

  1. Название скидки
  2. Код скидки в системе
  3. Ограничение на количество использования
  4. Метод подсчета скидки, суммы или процента
  5. Тип скидки
  6. Распространение скидки
  7. Временной интервал действия скидки

Примечание: вы можете создать скидку для каждой языковой версии магазина при помощи переключателя языковой версии, доступного в форме редактирования скидки

Шаг 4

Сохранить изменения при помощи кнопки “Создать” или “Сохранить и выйти” в правом верхнем углу экрана
