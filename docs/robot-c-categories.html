Работа с категориями

Что изменится на сайте

Главная страница

Шаг 1

Чтобы создать новые или отредактировать уже созданные товарные категории, следует:

Для создания категории

Каталог товаров → Категории товаров → Клик по кнопке “Создать категорию”

Для редактирования категории

Каталог товаров → Категории товаров → Клик по названию категории, подлежащей редактированию

Шаг 2

В открывшейся форме редактирования/создания следует указать/изменить соответствующие данные в полях:

  • Название
  • Родитель
  • URL
  • Сортировка по умолчанию

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти”

Если вам нужно заполнить только основные данные без расширенных настроек, рекомендуем попробовать функцию быстрого наполнения - быстрое создание категории товара
