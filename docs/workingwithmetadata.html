Работа с метаданными

Для того, чтобы указать метаданные странице, необходимо:

Шаг 1

Совершить переход в панель администратора в соответствии с типом настраиваемых страниц:

Страницы брендов - Каталог товаров → Бренды → клик по названию бренда, подлежащего настройке

Страницы товарных категорий/подкатегорий - Каталог товаров → Категории товаров → клик по названию категории/подкатегории, подлежащей настройке

Страницы товаров - Каталог товаров → Список товаров → клик по названию товара, подлежащего настройке

Информационные страницы - Страницы → Список страниц → клик по названию страницы, подлежащей настройке

Шаг 2

В открывшейся форме необходимо:

Для страниц брендов - заполнить поля внизу экрана:

  • Meta Title
  • Meta Description
  • Meta Keywords

Для страниц категорий/подкатегорий - заполнить поля внизу экрана:

  • H1
  • Meta Title
  • Meta Description
  • Meta Keywords

Для страниц товаров - Перейти на вкладку "Настройки"

И заполнить поля :

  • Meta Title
  • Meta Description
  • Meta Keywords

Для информационных страниц - перейти на вкладку "Свойства"

И заполнить (самостоятельно или воспользовавшись функцией “Автоподбор”) поля:

  • Meta Title
  • Meta Description
  • Meta Keywords

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти” в правом верхнем углу экрана.

Примечание: генерация метатегов для страниц по умолчанию автоматическая. Изменить данную настройку можно, осуществив переход: Система → Глобальные настройки → вкладка “Управление метатегами”.

Примечание: на тарифах Рremmerce, начиная с пакета STANDART, становится доступным специальный модуль “SEO-эксперт”, который позволяет существенно упростить работу с метаданными страниц. Более подробно об этом модуле читайте в соответствующей статье документации.

Примечание: вы можете задать метаданные для каждой языковой версии отдельно при помощи переключателя языковой версии, доступного в форме редактирования страницы, подлежащей настройке.
