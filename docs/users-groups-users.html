Пользователи

Чтобы создать пользователя или группу пользователей (роль), необходимо:

Шаг 1

В панели администратора осуществить переход: Пользователи → ...

Для пользователя - ... → Список пользователей

Для группы (роли) - ... → Управление правами доступа

Шаг 2

Далее в открывшейся форме необходимо:

Для пользователя

1. Осуществить клик по кнопке "Создать пользователя"

2. В открывшейся форме внести изменения в соответствующие поля:

- E-mail

- Пароль

- Роль

- Полное имя

- Телефон

- Адрес

1. Осуществить клик по кнопке "Новая роль"

2. В открывшейся форме внести изменения в соответствующие поля:

- Имя

- Описание

- Важность

3. Осуществить выбор доступных прав для группы на вкладках:

- Магазин

- Базовые

- Модули

Шаг 3

Сохранить изменения при помощи кнопки “Создать” или “Создать и выйти” в правом верхнем углу экрана
