Комплекты

Для того, чтобы создать новый набор или отредактировать уже существующий, необходимо:

Шаг 1

В панели администратора осуществить переход: Каталог товаров → Наборы товаров → …

Для создания нового комплекта: ... → Осуществить клик по кнопке “Создать набор”

Для редактирования существующего: ... → Осуществить клик по уникальному идентификатору набора в столбце “ID”

Шаг 2

В открывшейся форме создания/редактирования в соответствующих полях следует:

Для создания набора

  1. Указать главный товар в поле "Главный товар"
  2. Указать дополнительный товар в поле "Связанные продукты"
  3. Указать процент скидки на дополнительный товар
  4. Осуществить выбор в ячейках "Активный" и "Только для зарегистрированных пользователей"

Для редактирования набора

  1. Внести изменения (при необходимости) в поле "Главный товар"
  2. Внести изменения (при необходимости) в поле "Связанные продукты"
  3. Внести изменения (при необходимости) в поле "Скидка %"
  4. Внести изменения (при необходимости) в ячейках "Активный" и "Только для зарегистрированных пользователей"

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Создать” в правом верхнем углу экрана
