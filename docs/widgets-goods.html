Виджеты товаров

В системе Premmerce по умолчанию используются 3 виджета товаров, отображаемые в шаблоне соответственно используемому дизайну:

  • Виджет “Акционные товары”
  • Виджет “Популярные товары”
  • Виджет “Новинки”

Шаг 1

В панели администратора осуществить переход: Каталог товаров → Список товаров

Шаг 2

Далее, чтобы вывести ваши актуальные товары через виджеты товаров, в открывшемся списке следует присвоить товарам соответствующие отметки в столбце “Статус”

Отметка для акционных товаров

Отметка для популярных товаров

Отметка для новинок

Примечание: присвоить отметку товару можно непосредственно в форме создания товара после его сохранения

Шаг 3

Очистите кеш интернет-магазина, осуществив переход: Настройки → Очистить кеш
