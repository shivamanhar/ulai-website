Количество товаров

Для того, чтобы осуществить настройки количества вывода, необходимо:

Шаг 1

В панели администратора осуществить переход: Настройки → Настройки магазина → вкладка “Внешний вид”

Шаг 2

В части вкладки "Внешний вид" в соответствующих полях указать:

  1. Количество товаров, отображаемых на одной странице товарной категории
  2. Возможные значения для количества вывода товаров на странице (позже покупатель сможет выводить товар на страницу категории, осуществляя выбор значения из поля "На странице")

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти” в правом верхнем углу экрана
