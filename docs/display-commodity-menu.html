Меню категорий товаров

Для настройки отображения меню следует:

Шаг 1

В панели администратора осуществить переход: Модули → Управление дизайном → Вкладка “Колонки меню”

Шаг 2

Далее в открывшейся вкладке следует осуществить настройку отображения меню, оперируя значениями:

  1. Поля отображения уровней
  2. Колонки "Отображать в колонке"

Для отображения меню в виде колонок следует задать значения:

  • поля отображения уровней - значение “Все категории”
  • колонки "Отображать в колонке" - значение “0”

Для отображения меню в виде дерева категорий следует задать значения:

  • поля отображения уровней - значение “Категории второго уровня”
  • колонки "Отображать в колонке" - значение “0”

Для отображения меню в виде дерева с колонками категорий следует задать значения:

  • поля отображения уровней - значение “Категории второго уровня”
  • колонки "Отображать в колонке" - числовые значения от 0 до 10

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” в левом нижнем углу экрана
