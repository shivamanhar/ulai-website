Работа с товарами

Шаг 1

Каталог товаров → Список товаров

Для создания нового товара - кликаем по кнопке “Создать товар”

Для редактирования - кликаем по названию товара, подлежащего редактированию, в соответствующем столбце

Шаг 2

В форме создания нового товара указываем основные данные в разделе “Информация” в соответствующих полях:

  • Название товара (при необходимости название варианта)
  • Цена
  • Артикул товара (не обязательно)
  • Количество единиц товара в наличии
  • Валюта
  • Загрузить изображение
  • Бренд
  • Главная товарная категория (и при необходимости дополнительные товарные категории)
  • Краткое и полное описание

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Создать” в правом верхнем углу экрана

Шаг 4

Указать/изменить свойства товара на вкладке формы редактирования "Свойства"

Перед тем, как присваивать товару значения для характеристики, их нужно создать

Загрузить/изменить дополнительные изображения

Шаг 5

Сохранить изменения при помощи кнопки “Сохранить” или “Создать” в правом верхнем углу экрана

Если вам нужно заполнить только основные данные без расширенных настроек, рекомендуем попробовать функцию быстрого наполнения - быстрое создание товара
