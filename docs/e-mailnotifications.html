E-mail-уведомления

Чтобы изменить информацию в E-mail-уведомлениях, вам необходимо:

Шаг 1

Модули → Управление E-mail-уведомлениями

Шаг 2

Для внесение изменений в контактные данные магазина осуществляем клик по кнопке настройки, которая находится в списке E-mail-уведомлений

Шаг 3

В поле открывшейся формы редактирования укажите актуальные данные вашего магазина. Информация в текстовом поле выводится как обертка для всех писем магазина

В текстовом поле не удаляйте строчку $content, на ее месте будет добавляться содержимое всех писем!

Шаг 4

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти”
