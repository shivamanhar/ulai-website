Страница брендов

Шаг 1

Для создания бренда

Каталог товаров → Бренды → Клик по кнопке “Создать бренд”

Для редактирования бренда

Каталог товаров → Бренды → Клик по названию бренда, подлежащего редактированию

Шаг 2

В открывшихся формах редактирования/создания следует указать корректные данные в полях:

  • Категории товаров
  • URL
  • Описание
  • Изображение

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Создать” в правом верхнем углу экрана

Чтобы данный функционал работал корректно, необходимо:

  • Удалить все демо-товары
  • При создании актуальных товаров присваивать им соответствующий бренд в поле “Название бренда”

Если вам нужно заполнить только основные данные без расширенных настроек, рекомендуем попробовать функцию быстрого наполнения - быстрое создание бренда
