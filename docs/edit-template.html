Изменение дизайна

В нашем репозитории представлено множество шаблонов как платных, так и бесплатных. Подобрать из линейки шаблонов тот, который приглянулся вам более всего, и установить его на ваш интернет-магазин можно, осуществив ряд простых действий:

Шаг 1

В панели администратора осуществите переход: Модули → Управление дизайном → клик по вкладке “Список дизайнов”

Шаг 2

В открывшейся вкладке осуществите клик по кнопке “Добавить дизайн”

Далее произойдет перенаправление на наш официальный сайт.

Шаг 3

Ознакомьтесь с представленными дизайнами. Около каждого шаблона, представленного в репозитории, находится кнопка "Демо", клик по которой открывает его полноценную онлайн-демо-версию.

Шаг 4

Выберите дизайн, который вам понравился, и осуществите клик по кнопке “Купить” - если вы выбрали платный дизайн:

“Установить на интернет-магазин” - если вы выбрали бесплатный дизайн

Примечание: кнопки “Купить” или “Установить на интернет-магазин” находятся на странице шаблона, которая открывается при помощи кнопки “Подробнее”.

Шаг 5

Этот шаг будет зависеть от того, какой именно шаблон вы выбрали, - платный или бесплатный.

При выборе платного шаблона для приобретения дизайна следует осуществить следующее:

1. Выбрать удобный вам способ оплаты.

2. Оплатить заказ, который теперь будет содержаться на вкладке “Список дизайнов”, при помощи клика по кнопке “Оплатить”.

При выборе бесплатного шаблона после клика по кнопке “Установить на интернет-магазин” интересующий вас шаблон будет загружен автоматически и сразу же отобразится на вкладке “Список дизайнов” модуля “Управление дизайном” (в примере на скриншоте это дизайны Box Green и New Level, шаблон Light используется по умолчанию).

Однако новый шаблон еще не будет установлен. Для завершения установки нового шаблона следует:

1. Осуществить клик по кнопке “Установить”.

2. Прочитать и принять лицензионное соглашение.

3. Клик по кнопке “Установить”.

Все, новый дизайн установлен и уже используется в вашем интернет-магазине.
