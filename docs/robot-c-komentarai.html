Работа с комментариями

Шаг 1

Модули → Комментарии

Шаг 2

Для модерации

Осуществить клик по ID модерируемого комментария

При необходимости редактируем:

  • Имя пользователя
  • Е-mail пользователя
  • Статус
  • Текст комментария

Для настройки

Осуществить клик по кнопке "Настройки"

Указываем данные в форме:

  • Максимальная длина комментария
  • Частота комментирования
  • Отмечаем необходимые пункты в блоке ячеек

Шаг 3

Сохранить изменения при помощи кнопки “Сохранить” или “Сохранить и выйти” в правом верхнем углу экрана
