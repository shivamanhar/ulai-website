Быстрое наполнение

Быстрое создание категории товара

В панели администратора осуществите переход: Каталог товаров → Категории товаров → клик по кнопке “Быстрое создание”

Заполните необходимые поля и нажмите кнопку "Создать"

Быстрое создание товара

В панели администратора осуществите переход: Каталог товаров → Список товаров → клик по кнопке “Быстрое создание”

Заполните необходимые поля и нажмите кнопку "Создать"

Заполните необходимые поля и нажмите кнопку "Создать"
